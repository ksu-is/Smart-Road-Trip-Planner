# Smart-Road-Trip-Planner Description
I am creating this for my Application Development class. This software is made for those who are living the van life, road trip often, or have a friends road trop coming up. Features that I would like to include are; best routes to take, organizing stops along the wy, costs, managing shared expenses. In hopes to create route efficiency and to make the most of the time you have while on the road.
